Ráfaga Matos

Sou desenvolvedor web front-end com uma paixão por criar interfaces limpas e funcionais. Comecei minha carreira como designer gráfico, o que me deu uma forte base em design visual e uma atenção especial aos detalhes estéticos. Com o tempo, migrei para o desenvolvimento web, onde descobri a satisfação de transformar designs em websites interativos e responsivos. Utilizo HTML, CSS e JavaScript para construir interfaces intuitivas, e tenho experiência com frameworks como React. Gosto de desafios e estou sempre aprendendo novas tecnologias para aprimorar minhas habilidades e entregar produtos que unam design e funcionalidade de forma harmoniosa.
